# CitySpot

CitySpot is a mobile application developed with React Native and Expo for the Mobile Programming course.

The application allows users to discover nearby places such as cafés, restaurants, study places and shopping areas. Users can search places, view them on a map, save favorite places, rate places, upload photos and use an AI assistant.

## Technologies Used

- React Native
- Expo
- React Navigation
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- React Native Maps
- OpenStreetMap Overpass API
- OpenRouteService API
- OpenCage Geocoding API
- OpenAI API

## Environment Variables

API keys are not included in the repository for security reasons.

To run the project, create a `.env` file using the structure provided in `.env.example`.
